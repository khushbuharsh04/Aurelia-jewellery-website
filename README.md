# 💎 Aurelia Jewellery Website

Aurelia is a luxury jewellery e-commerce website developed using WordPress, Elementor, WooCommerce, and the Blocksy theme. The project focuses on creating a premium online shopping experience with a clean, elegant, and responsive design.

## ✨ Features

- Luxury and modern jewellery website
- Fully responsive design
- Product catalog
- Shopping cart
- Wishlist
- Custom CSS styling
- Elegant homepage and product pages

## 🛠️ Technologies Used

- WordPress
- Elementor
- WooCommerce
- Blocksy
- HTML
- CSS
